# 燕云百业战管理工具 - 部署说明

## 🎉 部署完成

服务器已成功部署并可以被局域网和公网访问！

## 📍 访问地址

### 本机访问
```
http://localhost:3000
```

### 局域网访问（同一WiFi下的设备）
```
http://192.168.1.7:3000
```

### 🌐 公网访问（固定URL）
```
https://yanyun-baiye.loca.lt
```

**特点**：
- ✅ 固定URL，不会变化
- ✅ 完全免费，无需注册
- ✅ 支持长时间请求（智能配队等功能）
- ✅ 自动HTTPS加密
- ⚠️ 首次访问时点击页面上的 "Click to Continue" 按钮即可

**启动方式**：
```bash
# 方式1：使用启动脚本（推荐）
./start-public.sh

# 方式2：手动启动
node server.js &
lt --port 3000 --subdomain yanyun-baiye
```

## 💾 数据存储说明

**重要**：所有数据现在都保存在服务器端，实现了真正的多人共享！

- ✅ 数据保存在服务器的 `data/` 目录中
- ✅ 所有访问公网URL的用户看到的是同一份数据
- ✅ 人才库数据：`data/players.json`（已包含45名玩家）
- ✅ 配队数据：`data/teams.json`（保存所有日期的配队记录）
- ✅ 数据持久化：只要服务器运行，数据就会一直保存
- ✅ 多人协作：任何人的修改都会实时同步到服务器

**数据特点**：
- 新建百业战日期后，数据会永久保存
- 历史配队记录可以随时查看和加载
- 所有用户共享同一个人才库
- 不会因为清除浏览器缓存而丢失数据

## 📱 使用方法

### 局域网内其他设备访问
1. 确保设备连接到同一个WiFi网络
2. 在浏览器中输入：`http://192.168.1.7:3000`
3. 即可使用所有功能

### 功能列表
- ✅ 人才库管理（添加、编辑、删除、OCR导入）
- ✅ 报名玩家管理（手动添加、OCR导入）
- ✅ 智能配队（AI自动分配）
- ✅ 拖拽式队伍调整
- ✅ 配队方案导出（图片格式）
- ✅ 历史记录查看

## 🔧 技术栈
- 后端：Node.js + Express
- 前端：原生HTML/CSS/JavaScript
- AI：Claude Sonnet 4.6 (OCR识别 + 智能配队)
- 图片处理：Sharp (压缩优化)

## 📝 注意事项
1. 服务器需要保持运行状态才能访问
2. 局域网IP地址可能会变化，如果无法访问请重新获取IP
3. 数据存储在浏览器本地，清除浏览器数据会丢失配队信息
4. 建议定期导出配队图片作为备份

## 🌐 公网访问（可选）
如需从外网访问，可以使用以下方案：
1. **ngrok**：快速创建临时公网隧道
2. **frp**：自建内网穿透服务
3. **云服务器**：部署到阿里云/腾讯云等
